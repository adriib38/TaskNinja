## Deployment

Pasos para iniciar la aplicación:

#### Instalar dependencias
```bash
  pip install -r requirements.txt
```

#### Generar base de datos con modelos
```bash
  python manage.py migrate
```

#### Iniciar aplicación
```bash
  python manage.py runserver
```
