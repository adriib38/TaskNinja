TaskNinja {API}

Work with our API four GET, POST and DELETE your Tasks and Projects.

You need to get your personal token and use it in the request header.

You can use the Postman application to test the API.

Your personal TOKEN API:

In the any request you need to add the header:

Authorization: Token {{ user.auth_token.key }}
